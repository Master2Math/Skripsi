# Skripsi
Kinerja jaringan yang optimal menjadi kebutuhan penting dalam mendukung
kegiatan administrasi dan pembelajaran di lingkungan pendidikan. MIN 3 Pontianak
menghadapi kendala berupa penurunan kualitas jaringan akibat penggunaan satu jalur
bandwidth untuk seluruh pengguna yang menyebabkan trafik data tidak terkelola
secara efisien. Penelitian ini bertujuan untuk mengoptimalkan Quality of Service (QoS)
jaringan melalui penerapan manajemen bandwidth menggunakan metode Queue Tree
pada router. Metode yang digunakan meliputi analisis kebutuhan jaringan,
perancangan pembagian trafik berdasarkan kategori pengguna, konfigurasi parent dan
child queue, serta pengujian performa jaringan menggunakan parameter throughput,
latency, jitter, dan packet loss. Hasil implementasi menunjukkan bahwa pembagian
alokasi bandwidth ke dalam beberapa kelas layanan seperti kelas ujian dan kelas guru
mampu meningkatkan efisiensi penggunaan bandwidth serta meminimalkan gangguan
antar pengguna. Pengujian QoS memperlihatkan peningkatan throughput, penurunan
latency, serta jitter yang lebih stabil dibandingkan kondisi sebelum penerapan. Dengan
demikian, metode Queue Tree terbukti efektif dalam mengoptimalkan kualitas layanan
jaringan di MIN 3 Pontianak.
Kata Kunci: Queue Tree, Quality of Service, Manajemen Bandwidth, QoS, Mikrotik

Optimal network performance is essential to support administrative and learning activities in educational institutions. MIN 3 Pontianak experienced network degradation due to the use of a single bandwidth channel shared by all users, resulting in inefficient traffic management and uneven bandwidth distribution. This study aims to optimize network Quality of Service (QoS) through bandwidth management using the Queue Tree method on a MikroTik router. The research method includes network requirement analysis, traffic classification based on user categories, Queue Tree configuration, and performance evaluation using throughput, latency, jitter, and packet loss parameters. The implementation results show that bandwidth allocation into several service classes, such as examination and teacher classes, improves bandwidth efficiency and minimizes interference among users. QoS testing indicates increased throughput, lower latency, and more stable jitter compared to the previous network condition. Therefore, the Queue Tree method is effective in optimizing network service quality and improving network performance at MIN 3 Pontianak.
